## Purpose

Gunakan skill ini ketika mereview hasil AI Coding Agent.

---

# Review Checklist

## Architecture

Periksa:

```txt
Folder structure sesuai
Module structure sesuai
```

---

## Backend

Periksa:

```txt
Controller tipis
Service berisi business logic
DTO digunakan
Permission digunakan
Ownership digunakan
```

---

## Database

Periksa:

```txt
Index ada
Unique ada
Relation benar
```

---

## Frontend

Periksa:

```txt
Page bersih
Service terpisah
Store terpisah
```

---

## Security

Periksa:

```txt
Tidak expose private data
Tidak bypass permission
Tidak bypass ownership
```

---

## Typescript

Periksa:

```txt
No any
No duplicate type
```

---

# Final Verdict

Berikan hasil:

```txt
PASS
PASS WITH IMPROVEMENT

```
